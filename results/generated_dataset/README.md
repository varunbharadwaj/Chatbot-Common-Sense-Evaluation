## Format of Dataset:

1. dataset: Query \_\_eou\_\_ negative-response
2. candidates: 
	Query \_\_eou\_\_ original-response [line 1]
	generated negative candidates [starts on line 2]
	[blank line - separator/end of generated candidates]
